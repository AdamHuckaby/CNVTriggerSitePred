#######DEPENDENCIES#######

Has only been tested on Ubuntu 18.04 but should work with the following dependencies correctly installed:
1) Bash
2) R statistics software
3) python3
4) Vienna RNAfold
5) EMBOSS software package for molecular biology

-------

#######INSTALLATION#######

1) Install R. Default installed for Ubuntu 18.04 is R version 3.4.4 (2018-03-15) -- "Someone to Lean On"
    sudo apt-get update
    sudo apt-get install r-base r-base-dev

2) Check python version

3) Install EMBOSS software package. Check it is correctly installed by running shuffleseq
    sudo apt-get install emboss
    shuffleseq

4) Install Vienna RNAfold using pre-compiled binary for Ubuntu 18.04 from
https://www.tbi.univie.ac.at/RNA/#
